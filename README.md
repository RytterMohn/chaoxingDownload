# chaoxingDownload 超星章节内ppt下载

作者：西电网信院的废物rytter

按下d或者点击下载按钮可弹出该ppt的网页，点击网页保存即可下载

有问题联系ryttermohn@gmail.com

本脚本可在 Edge浏览器101.0.1210.39 (正式版本)下运行
